# VPN Pro for Samsung Galaxy A04e

Professional VPN application running on Termux for Samsung Galaxy A04e.

## Features
- 12 Free VPN servers worldwide
- WireGuard & OpenVPN support
- Advanced speed test
- Kill switch for privacy
- Arabic UI
- Optimized for Galaxy A04e

## Quick Install
```bash
pkg update && pkg upgrade -y
pkg install git -y
git clone https://github.com/king56222/vpn-pro-galaxy-a04e.git
cd vpn-pro-galaxy-a04e
chmod +x scripts/install.sh
./scripts/install.sh
./scripts/start.sh
