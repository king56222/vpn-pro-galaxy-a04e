# المساهمة في المشروع

شكراً لاهتمامك بالمساهمة في مشروع VPN Pro! 🎉

## كيفية المساهمة

### 1. الإبلاغ عن مشكلة
- استخدم GitHub Issues
- اشرح المشكلة بالتفصيل
- أضف لقطات شاشة إن أمكن
- اذكر إصدار التطبيق والجهاز

### 2. اقتراح ميزة جديدة
- اشرح الميزة وفائدتها
- قدم أمثلة على الاستخدام
- اذكر كيف تفيد المستخدمين

### 3. تحسين الكود
- Fork المشروع
- أنشئ فرعاً جديداً: `git checkout -b feature/amazing-feature`
- نفذ التغييرات
- تأكد من اجتياز الاختبارات
- أرسل Pull Request

## معايير الكود

- استخدم Python 3.8+
- اتبع PEP 8
- أضف تعليقات للكود المعقد
- اكتب اختبارات للوظائف الجديدة
- وثق API الجديدة في API.md

## إرشادات الـ Pull Request

1. استخدم عنوان واضح
2. اشرح التغييرات بالتفصيل
3. أشر إلى المشكلة المرتبطة
4. تأكد من اجتياز CI/CD
5. انتظر المراجعة

شكراً لمساهمتك! 🌟
